# Contribution Guide

## Install dependencies

This repo uses bun as its package manager.

```console
bun install
```

## Pull Requests

If you send pull request, please pass the tests with the following commands.

```console
npm run build
npm run test
npm run lint
npm run check
```

## Useful tools

#### Changesets

```console
npm run changesets
```

#### Benchmark

```console
npm run benchmark
```

#### Formatter

```console
npm run format
```
